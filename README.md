# DriveCloud - Frontend

A modern cloud storage web application for uploading and managing files and folders.

## Tech Stack

- **Framework:** React with Vite
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Build Tool:** Vite

## Features

### Current Features
- ✅ **File Upload:** Upload files to cloud storage
- ✅ **Folder Management:** Create and organize folders
- ✅ **Star Folders:** Mark important folders for quick access
- ✅ **Share Folders:** Share folders with others

### Pages
- Home/Dashboard
- Starred (Coming Soon)
- Shared (Coming Soon)

## Notes

- File sharing feature is currently limited to folders only
- Starred and Shared pages are under development
- Storage powered by AWS S3

## Getting Started

```bash
npm install
npm run dev
```
